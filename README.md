## Code Challenge
A Star Wars character selector using the Starwars API https://swapi.dev/

## Installation
Clone this repository

### Run the Express Back-end
1. Navigate to the starwars-api directory

```cd starwars-api```

2. Install the Node Modules

```npm install```

3. Run the project

```npm run start```

### Run the Front-end Angular project
These steps are almost the same as the back-end project. If you do not wish to run them as background processes, you will need to use 2 terminals 
1. Navigate to the starwars directory

```cd starwars```

2. Install the Node Modules

```npm install```

3. Run the project

```npm run start```

4. In a browser, navigate to http://localhost:4200/

## Usage
Use the bottom menu buttons to select a character.
